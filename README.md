## Running IHaskell

Start Docker container by running `.\start.sh`. Open http://localhost:8888?token=x in browser.

## Commands

Stop the container

    docker stop ihaskell_notebook

Pause the container

    docker pause ihaskell_notebook

Unpause the container

    docker unpause ihaskell_notebook
